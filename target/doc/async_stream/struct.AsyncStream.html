Produce items for a stream from an async closure.

AsyncStream::new() takes a Future (async closure, usually), and returns an AsyncStream that implements a Stream.

Async closures are not stabilised yet, but you can wrap an async block in a closure which is very similar, as documented in the async/await RFC.

Example:

ⓘThis example is not tested
let mut strm = AsyncStream::<u8>::new(|sender| async move {
    for i in 0u8..10 {
        sender.send(i).await;
    }
});
